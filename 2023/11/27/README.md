```
pip install sanic blacksheep uvicorn aiohttp uvloop
```

Then run either

```
python server.py
```
for the sanic server or

```
python blackserver.py
```
for the blacksheep server  or

```
python aioserver.py
```
for the aiohttp server.

Then in another process run

```
python client.py
```

Be patient.
